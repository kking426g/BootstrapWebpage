Class List

.row
- create a row <div class = "row">. Then, add the desired number of columns 
<div class="row">
<div class="col-sm-4">.col-sm-4</div>
<div class="col-sm-4">.col-sm-4</div>
<div class="col-sm-4">.col-sm-4</div>
</div>
.col-sm-4
- create a row <div class = "col-sm-4">. Below we have collected some examples of basic Bootstrap grid layout 
<div class="row">
<div class="col-sm-4">.col-sm-4</div>
<div class="col-sm-4">.col-sm-4</div>
<div class="col-sm-4">.col-sm-4</div>
</div>
.img-rounded
- create a row <img class = "img-rounded">. Adds rounded corners to an image (IE8 does not support rounded corners) 
<img src="cinqueterre.jpg" class="img-rounded" alt="Cinque Terre" width="304" height="236">
.btn .btn-info
- create a info button <div class = "btn btn-info">. Apply a light blue color button 
<button type="button" class="btn btn-info"></button>
.btn-sm
- create a small button <div class = "btn-sm">. Bootstrap provides four button sizes, lg, md, sm, xs 
<button type="button" class="btn btn-info btn-sm"></button>
.jumbotron
- create a jumbotron <div class = "jumbotron">. if you want the jumbotron to extend to the screen edges 
<div class="jumbotron">
<h1>Bootstrap Tutorial</h1>
 <p>Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive,mobile-first projects on the web.</p>
</div>
